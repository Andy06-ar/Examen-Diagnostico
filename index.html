Conceptos Clave del Diseño Web

¿Que es el diseño web?

Se conoce como desarrollo web al proceso de crear y mantener un sitio web que sea funcional en internet, a través de diferentes lenguajes de programación, según el modelo y la parte de la página que corresponda. Cada sitio tiene una URL única que lo distingue de los demás en la red informática mundial. Un sitio web puede clasificarse de diferentes formas. Para cuestiones de desarrollo web principalmente se divide en dos partes.

  • Frontend: Es la parte que interactúa con el usuario, tanto en imagen como en función. Por ello está íntimamente relacionada con la experiencia del usuario (UX) y la interfaz de usuario (IU).
  • Backend: Se refiere a la parte que está en contacto directo con el servidor; es donde se aplica el código de programación para crear la estructura. Permanece en un segundo plano a cargo de la accesibilidad, actualización, bases de datos y cambios del sitio.

Que es el internet?

El término Internet se utiliza para hacer referencia a una red de computadoras interconectadas entre sí a nivel mundial con el objetivo de hacer común información de acceso público.

Que significa WWW?

La WWW (World Wide Web), que es una colección de documentos y recursos accesibles a través de Internet, mediante direcciones URL (Uniform Resource Locator).

En donde se hacen las páginas web?

En HTML (HyperText Markup Language) para estructurar el contenido. CSS (Cascading Style Sheets) para dar estilo y formato, mientras que JavaScript añade interactividad.

Que es un servidor?

Es un equipo o software que proporciona servicios a otros dispositivos en la red, en estos se alojan las paginas y sitios web dentro de centros de datos.

Que es HTTP?

Es el protocolo que se utiliza para transferir páginas web desde los servidores a los navegadores web, los cuales permiten visualizar y navegar en las paginas web.

Que es ICANN?

(Internet Corporation for Assigned Names and Numbers) es la organización encargada de la gestión y coordinación del sistema de nombres de dominio y direcciones IP en Internet.

Generaciones de la Web

  • WEB 1.0: Primera generación de la web, estática y de solo lectura.
  • WEB 2.0: Segunda generación de la web, con interacción y contenido generado por usuarios.
  • WEB 3.0: Tercera generación de la web, con mayor enfoque en datos y semántica.
  • WEB 4.0: Cuarta generación de la web, centrada en la integración de inteligencia artificial.WEB 5.0: Quinta generación de la web, con énfasis en la interacción emocional y la inteligencia artificial avanzada.
